# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

- Email: pileroid@gmail.com
- Subject: [Security] Vulnerability Report
- Include:
    - A detailed description of the issue
    - Steps to reproduce it
    - Version of the project
    - Any screenshots or logs that help illustrate the problem

We take all reports seriously and will respond as soon as possible.

## Supported Versions

This project is actively maintained. Security fixes will be applied to the latest version and supported releases.
